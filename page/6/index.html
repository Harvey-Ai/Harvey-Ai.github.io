    CentOS安装过程遇到的引导问题

    2014-02-08

    寒假回来第一件事情就是给家里电脑更新换代，给家里电脑装了CentOS6，没想到整整搞了一个寒假，直到最后一天才安装完毕。

    安装CentOS

    家里没有Linux光盘，没有U盘，只能用硬盘安装的方法，我这里使用了修改grub引导的方法进行安装，原理很简单，电脑的头分区是一块叫mbr的扇区，一共512Byte，记录了启动的信息和分区表信息，修改这里并让电脑先引导grub便能引导linux安装程序。Fedora15以上版本的系统用的是OPT引导方式，家里机器安装不上，于是换更低的版本，没想到还搞不定，可能是分区格式是ntfs的缘故，最后选择装了CentOS，后来还使用了在线安装的方法。

    Srm595 DVI2

    2013-12-18

    1. (DIV2)1000分题，题目大意：给定A, B, C，问存在多少对(X, Y, Z)满足条件0 =< X <= A，0 =< Y <= B，0 =< Z <= C 并且 X ^ Y <= Z。其中A, B, C不大于10e9。

    解题思路：首先枚举X, Y是不现实的，n^2的复杂度太大了，于是需要找这个问题的重叠性。可以看到，数字的前缀是可能重复的，如果能知道所有N+1位前缀的答案个数，就能递推出某个N位前缀的答案数，分四种情况计算。例如所有前缀为123X的X，前缀为234X的Y的答案对数知道，那么以123和234为前缀X，Y的答案对数可以分四种情况计算（列举2种）。

    明天，昨天，今天

    2013-12-03

    一直想写一些文字激励一下自己，可是不知不觉中总是以各种理由推迟了下来，一直到现在。 最近一些时间对自己和周围的世界在不断的碰壁中有了更好的认识，现在回顾一下我自己，以及以后的目标。

    首先，我的未来是什么样子的，想闯去创业，还是安居乐业去公司做一颗螺丝钉？年轻的时候雄心勃勃，一心想创业赚大钱，可是把事情想的太简单了，以为把技术搞好了就能创业，就能技术革新。现在看来，这是不现实的，现在能做成的任何事情都是能满足人的欲望的，只有能做出进一步满足人欲望的东西才能有成功，而技术更新只是在欲望驱使下的必然结果，但是能做出革新的事情可不是一天两天的事情，要有坚持不懈的毅力和不怕挫折的探索精神（像蛋总一样）。但是，如果只是想赚钱的话，直接创业显然是不适合的，现代社会是经济社会，要赚钱那最少得先有钱，所以空手创业就是痴人做梦，是不可能成功，所以要能走一条可持续发展之路就必须要先就业，先有钱，在一点一点积累下来，然后再走下一步。
