C语言程序设计一

2014-07-18 | Language

K & R在1972年创造了C语言，没想到时间过去快半个世纪了，C语言依然流行，在TIOBE的语言排行榜上一直稳坐前两把交椅，不得不说C语言是非常美丽的语言。最近时间比较充裕，所以把经典的《C programming language》看了一下，回味无穷，比以前看的某国内出版名作好的不止一点两点。于是，对C语言重新做一下总结。

概况

C语言的发明就是为了编写Unix，用C重写了Unix之后，Unix取得了巨大的成功。而C语言由于其继承了Unix的思想——简单、高效、可移植性强，同样也被广泛的使用。C是比较底层的语言，也被称为“系统语言”。C语言不是学院派的产物，其思想与Unix非常相似，就是“实用、简单”。

[caption id=”attachment_750” align=”alignnone” width=”379”] 《C programming language》[/caption]

到目前为止，C语言一共经历了4代，分别是第一版、C89、C99、C11，每个版本号的数字后缀是标准发布的年份，这本《C programming language》是基于C89标准撰写的，而平时大家所说的“ANSI C”指的就是这个版本。C89对第一版进行了修改，除了对声明进行调整外，最大的修改部分是：在函数声明时增加参数的类型声明，从而让编译器变得更加聪明。

不像Java，C语言非常简洁，规则少。像Unix一样，C也没有采取保护措施，程序员对程序有绝对的控制权，灵活到能控制操作系统的所有资源，按照K & R的说法，C的哲学是：“程序员知道他在做什么。”
C的特点如下：

  • 强类型，编译语言
  • 单线程、不提供任何垃圾回收机制和系统调用
  • 与操作系统关系紧密，被称为系统胶水

** 表达式与基本类型**

任何语言都是由一定的规则+数据组成，就好像自然语言中的名词和动词一样，只不过自然语言更加复杂。在C语言中，最基本的表达形式是表达式，基本表达式由操作数和操作符组成，操作数分为常量和变量两种，而操作符用于链接不同的操作数对象。

操作数

C语言中的任何操作数都有类型，不管是常量还是变量，当使用变量时，需要提前声明变量的类型。本质上说，C只有两种基本类型：整数类型和小数类型，甚至能将所有操作数都封装成整数（需要通过值针）。

既然C强制指定了每一个操作数的类型，就难免会发生类型转换，转换的原则是：向值域更加宽的类型转换。类型转换有隐式和显式两种，显式转换由程序员显示指定，以 type (var) 的方式将变量_var_转化成_type_类型。而隐式转换在程序中隐式发生，常常在比较、赋值运算、传参、返回值中发生，所以要注意一下隐式转换的发生。例如：条件表达式会根据后两个参数共同决定表达式类型。在下面的代码中，便存在隐式转换，输出分别是4和8，可以看出第二行代码中的常量1被转化成了浮点类型。

printf("%d\n", sizeof(1 > 0 ? 1 : 0));     // 输出4
printf("%d\n", sizeof(1 > 0 ? 1 : 0.0));   // 输出8

常量，即不能更改的量，常量也有类型，通过常量后缀指定，默认整数常量类型是int，默认浮点常量类型是double，后缀L、H、F、U 分别表明变量的长类型、短类型、单浮点和无符号属性。在对操作数估值处理（evaluate）时，常量被放在了编译阶段，而变量则必须在运行期。

操作符

C的操作符可以分为：赋值操作符、关系操作符、逻辑操作符、算术操作符、位操作符等，每个操作符都有相应的优先级和结合方向。最高优先级的操作符有四个，分别是：[], (), ->, .，结合方向由左到右，然后是一元操作符，包括：!, ~, ++, –, +, *(解引用）, &, sizeof，最低优先级的操作符则是逗号操作符。

*++p  /* 等于*(++p)， 增加指针后取内容 */
*p[0] /* 等于*(p[0])，先取数组中存的地址，再取该地址中的内容 */

在使用操作符时，需要注意“副作用（side effect）”的产生，副作用通常出现在操作数会相互影响的表达式中，这样的表达式一般含有函数调用、递增、递减操作以及嵌套表达式。下面两个表达式都可能出现副作用。

a = f() + g();
a[n] = ++n;

sizeof是一个特殊的一元标示符，结果会在编译时产生，所以如果在预编译中不能使用sizeof。

类型声明

C语言规定，变量函数都必须要声明。声明时除了指定类型，还要指定属性和存储类型，存储类型（storage-class-specifier）一共5种，类型（type-specifier）12种，限定符（type-qualifier）2种。

存储类型（storage-class-specifier）：

typedef int (*funPointType)(int, int);    /* funPointType 是一个函数指针的别名 */
funPointType fp;                          /* 用funPointType定义一个函数指针 fp */

类型（type-specifier）：int， bool，char等等。

限定符（type-qualifier）：（1）const：指定变量不容许被改变。（2）violate：跟编译器优化有关。

可见域

可见域值的是变量的词法可见性（lexical scope）和链接性（linkage）。词法可见性与编译有关，如果程序违反词法可见性的约束，会造成编译失败，而链接性则与链接有关，对满足则会链接失败。

词法可见性（lexical scope）规定了变量在代码中的可见范围，以函数为界限将变量分为：内部变量（internal）和外部变量（external），内部变量只在函数内部可见，而外部变量则可被编译单元可见，同时与外部变量名相同的内部变量会屏蔽外部变量。

链接性（linkage）则告诉变量在编译单元之间的可见性，若通过extern则指定有外部链接性，static则表明内部链接性。具有外部链接性的变量可以和外部编译单元中的变量进行共享，而具有内部链接性的变量仅具有内部链接性，外部不可见。

控制流程

由操作符和操作数组成的表达式构成了语言的基本元素后，相当于有了做菜材料，还需要做菜的厨艺，在这里相当于控制流程。C是结构化编程语言，提供结构化编程的基本流程控制，虽然在语法上还提供goto，但不提倡使用。dijkstra老人家很早以前就写过《Goto有害论》，所以就不要使用goto了吧。结构化编程能保证能构建一切程序的控制流程。

C只提供了3种类型的控制流：分组（grouping），选择（selection）和循环（loop）。

  • 分组用大括号{}实现，也可成为块。
  • 选择语句有：switch, if…else
  • 循化语句有：for, while, do…while
    此外，C还提供break、continue，方便程序中的单次忽略和跳出。
