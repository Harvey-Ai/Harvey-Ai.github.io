函数内存传递

2009-08-06 | Language
void GetMemory(char *p){
    p = (char *)malloc(100);
}

void Test(void)
{
    char *str = NULL;
    GetMemory(str);
    strcpy(str, "hello world");
    printf(str);
}

如果你写了这个程序，那么一定会崩溃。曾经一度以为是malloc申请的空间生存期只是在函数内，被收回了，其实是被某书被忽悠了！!

malloc申请的是堆空间，C语言中堆空间是必须要手动释放才能被释放的，所以malloc的100大小的空间没有释放。但是为什么不对呢？其实是C语言在参数传递时，会复制一个参数变量，如果传值变量p，那么函数会复制一个_p,这也就解释了C是值传递的原因了。本程序中是值传，所以p并未改变。

内存传递可以用双指针，或者是引用。
