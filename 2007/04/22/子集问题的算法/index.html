子集问题的算法

2007-04-22 | Algorithm

算法一：

递归。

n个数的子集包含两种情况：

1，包含某个数。2，不包含这个数；

int n;
int arr[100];
void dfs(int pos)
{
    int i, temp;
    if(pos == n)
    {
        for(i=0;i<n;i++)
            if(arr[i])
                printf("%d ",arr[i]);
        putchar('\n');
    }
    else
    {
        dfs(pos + 1);
        temp = arr[pos];
        arr[pos] = 0;
        dfs(pos + 1);
        arr[pos] = temp;
    }
}

算法二：

将子集映射为2进制数值。

将集合中将要出现的元素按顺序排列，每一个子集都是全集中某些元素出现，而某些元素不出现，那么将出现对应位设为1，没出现位设为0，那么，每一个子集合就对应一个二进制数值，这个数值最小为0，最大为2^(n-1)。
所以，把从0到2^(n-1)的二进制的对应集合输出就可以了。
