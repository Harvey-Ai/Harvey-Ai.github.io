JavaScript高级程序设计（上）

2013-12-13 | Language

JavaScript是一门神奇的语言。JavaScript属于脚本语言，但有更多的内涵，设计起初是为了简单地验证浏览器的客户端，不过现在的JavaScript不仅能做客户端还开发服务器端。总而言值，由昔日的浏览器巨头NetScape提出的Javascript是一款值得学习和使用的语言。接下来逐步揭开他的面纱。

JavaScript的今世前生

JavaScript第一次出现是在网景公司1995年发布的NetScape 2.0中，它被用来给在浏览器客户端验证表单的功能，功能十分简单，但是却能极大减轻服务器与客户端通讯负载。诈一听JavaScript，感觉好像和Java有一些关系，其实不然，有那么一点点关系，就是当时为了炒作搭上了Java那波热潮，将LiveScript被命名成了JavaScript。从语言上说，JavaScript是弱类型动态语言。

另一方面，除了网景公司，微软和其他公司都紧随其后地发布自己JavaScript的实现版本。最初有微软的JScript，然后各个版本遍地开花，于是标准化组织出现了，ECMA（欧洲计算机制造协会）对JavaScript搞了一套ECMAScript标准，从发布的标准从1.0开始，一直到现在的5.0版本。

JavaScript的概况

广义上的JavaScript包括三个部分，包括：ECMAScript、DOM、BOM。其中ECMAScript是核心，描述了JavaScript的语法、语句、关键字、操作符、对象等；DOM是DHTML技术的产物，是将XML文档映射成层次结构的树并提供操作接口的东西，这样就可以动态修改HTML元素了。BOM是浏览器本身的接口规范，通过这个接口用户可以调节窗口大小新开网页等等。前面两个比较重要，BOM直到HTML5才被纳入到规范中来的。

由于JavaScript有多个实现版本，所以不同的浏览器对JavaScript的支持是不相同的，也不能说那家支持的最好，只能说哪些特性被支持了，毕竟技术和标准都一直在发展。ECMAScript已经到5.0了，现在的大部分浏览器都支持了5.0的ECMAScript（插一句，ECMAScript5.0其实是3.0的回滚版，由于4.0版改动太大，后来被ECMA否决了），最新的DOM3标准除了修改元素界面还有load、save等，各浏览器对DOM的各个标准的支持程度不太相同。

ECMAScript基本语法

JavaScript的语法是类C语法，分号是分隔符，大括号包围语句块，区分大小写，开头需要是字母、美元号，并且注释也很相似。变量有六种，包括五种基本变量和一种引用变量，变量的类型是动态的、内存是不可见的，容易使用。

操作符和C语言也是非常相似的，包括布尔操作符、关系操作符、条件操作符、位操作服等。

ECMAScript作用域

  • ECMAScript的作用域当然有局部和全局之分，通过Var定义的变量是局部变量，直接使用的变量会是全局变量（JavaScript有一个作用域链，会一直找变量声明，如果找到最顶端还没有找到应该是添加一个变量，所以就变成了全局变量）。

  • 变量的的作用域是以函数为单位，没有块级作用域，这点和C是不一样的。

  • 在作用域里有作用域链的概念，局部的作用域可以通过回溯发现父作用域中的变量，反过来却是不行的。

  • 离开作用域，局部变量会被销毁。

  • 在with和try语句中，作用域会被延长。
    ECMAScript变量

  • 变量分为两种类型变量，五种简单类型变量，简单类型变量是存储的是实际值，拷贝时直接拷贝内容，另一一种是引用类型变量——对象，对象变量保存的是对象的引用，拷贝只会传递变量的指针。

  • 基本对象是存在栈内存中，引用变量存在堆内存中。

  • 垃圾收集有两种方式，引用计数法和标记法；前者是因为微软的DOM中有COM的实现，所以需要记录引用来管理内存，容易出现循环引用的状况（如A = B， B = A；实际二者都不在使用，但都有引用）；后者是记录不在使用的内存，局部变量在离开作用域即被标记待清理，全局变量也存在循环引用的状况，需要使用解引用法解决。

  • 变量定义有两种，一种是New + Object，另一种是字面表示法用{}包裹，后者看起来更像是封装。
    ECMAScript函数

  • 函数也是变量，分为函数声明和函数变量两种，当使用函数声明时，解析器存在函数声明提前。

  • 每个函数中有默认的this和callee对象，分别指向调用者和函数本身，同时还有一个arguments对象保存函数的参数列表，修改arguments会改变参数值，反之不会，所以说影响是单向的。

  • 可以使用用call调用函数，这样扩展了函数的作用域，如functionA.call(windows)，在windows域中调用该函数，即this是windows。

  • 数组也对象，并且数组內元素可以是不同类型。
    ECMAScript常用内置函数

  • push，pop，shift，unshift用于数组尾部压入\弹出，头部压入\弹出。

  • splice(pos, n, “add string” …)数组中删除或添加元素。

  • slice提取子数组。

  • toString字符串转换。

  • String的trim可以出去头尾空格字符， toLowerCase和toUpperCase转换大小写，substring取子字符串。
    在HTML中使用JavaScript

  • 使用元素<script></script>，有一些属性可以指定如：defer，async指定异步等。

  • HTML有严格模式和普通模式，不过HTML 5的头比较简单<!Doctype html>，用严格模式请用 use strict。

  • HTML不区分大小写。
