二维平面上构成任意多边形的必要条件

Mar 19, 2015 | Algorithm

题目：有一个二维平面，某人起初站在（0， 0）点，这个人可以朝任意方向走去，不过只能沿一个方向走，走到一定距离后才能换方向，现在，给一些距离vector<double> dist，每一个距离值只能被用一次，问这个人是否可以走到（x， 0）点？

分析：题目的实质上就是问一个封闭的多边形（无论凸还是凹）是否能由一些指定的边长构成。这道题目是我在一次SRM上碰到的，当时是过了，不过是用的形象思维，想的是如果这些边是一根根木条，然后推导的。今天在群里面问道的时候，发现思路有点混乱，所以在这里把思路整理，对结论进行一下证明。当满足一下两个条件时，可以构成封闭多边形：

  • 如果这个人一直朝x的方向走，如果还不能通过（x， 0）点，那么必不能走到（x， 0），条件一是：Sum(dist) > = x。
  • 如果其中有一段很长的距离，如果先按这个距离走了，然后其他距离都是朝（x， 0）走，然后也不能穿过（x， 0），那么怎么走都不可能到达（x， 0）。
    这里用一个循环不变式进行证明，先对这些距离从大到小排序，然后每次拿出最大的距离走，让其满足下面两个条件：(dist1, dist2 .. 是从大到小待走的距离，y是已经走的终点和最终终点的距离）
  1. dist1 + dist2 + dist3 + dist4 + … >= y
  2. dist1 <= dist2 + dist3 + …. + y => y >= dist1 - dist2 - dist3……
    然后走dist1后，y值的变化是怎样的呢，如果按最终终点反方向走，则newY = y + dist1；如果朝最终终点走，newY = abs(y - dist1)，所以newY 取值是[abs(y - dist1), y + dist1]。

然后是由上式的1， 2可以得出abs(y - dist1) <= dist2 + dist3 + ….，所以newY又满足1; 然后显然y + dist1 >= dist2 - dist3 - …，所以newY又满足2。循环满足以上直到最后形成三角形。证毕。
